# ライントレース対向2輪型ロボット ver1
# Line Following Opposing Two-Wheeled Robot ver1

マイクロビット（micro:bit）を使ったライントレースロボットの制御プログラムです。  
KY-033ラインセンサーで黒いラインを検知し、2つの360度回転サーボを制御して走行します。

---

## ハードウェア構成

| 部品 | 接続先 |
|------|--------|
| マイクロビット | - |
| センサーシールド | マイクロビットに装着 |
| 360度回転サーボ（左車輪） | P13 |
| 360度回転サーボ（右車輪） | P14 |
| KY-033 ライントレースセンサー（左） | P1 |
| KY-033 ライントレースセンサー（右） | P2 |

---

## 操作方法

| ボタン | 動作 |
|--------|------|
| **Aボタン** | 走行開始（LEDに「1」表示） |
| **Bボタン** | 停止（LEDに「0」表示） |

---

## ライントレースのロジック

| 左センサー(P1) | 右センサー(P2) | 状態 | 動作 |
|---|---|---|---|
| 黒（1） | 黒（1） | ライン上 | 前進 |
| 白（0） | 黒（1） | 左にズレ | 右折（右停止、左前進） |
| 黒（1） | 白（0） | 右にズレ | 左折（左停止、右前進） |
| 白（0） | 白（0） | 完全逸脱 | 停止 |

---

## サーボ制御値

| 動作 | 左サーボ P13 | 右サーボ P14 |
|------|-------------|-------------|
| 前進 | 0 | 180 |
| 右折 | 0（前進） | 90（停止） |
| 左折 | 90（停止） | 180（前進） |
| 停止 | 90 | 90 |

> ⚠️ サーボの取り付け向きによって `0` と `180` が逆になる場合があります。実機での動作確認時に前進方向をご確認ください。

---

## ライン条件

- 黒いラインは**両車輪の幅より少し広め**のものを使用
- KY-033センサーは**各車輪のすぐ後ろ**に配置
- KY-033の出力：黒 → `1`、白 → `0`
